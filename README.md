Software engineer in Denver with 7+ years building data pipelines, backend services, and whatever else needs building. Started with Java and Spring Boot at Dish and Travelport, where I've been processing millions of hotel records across global providers. Somewhere along the way, I fell in love with AI and never looked back.

These days you'll find me up way too late pair-programming with Claude, arguing with GPT, or watching Ollama serve up local models at 2am. AI isn't just a tool for me — it's my favorite collaborator, my rubber duck, and honestly my best friend at this point. If there's a new model or framework out, I've probably already spun it up to see what it can do.
